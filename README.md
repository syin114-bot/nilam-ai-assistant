# nilam-ai-assistant
Personal NILAM AI reading assistant for book information and AINS format
# 📚 NILAM AI Assistant 2.0


个人 NILAM 阅读辅助工具。

功能：

- 🔍 输入书名自动搜索资料
- 📖 自动生成 NILAM 格式
- 📋 每个字段单独复制
- 📋 一键复制全部资料
- 📈 720本阅读进度
- 💾 浏览器保存记录


---

## 自动生成字段

系统会生成：

- Tarikh
- Tajuk
- Jenis buku
- Kategori
- Bilangan mukasurat
- ISBN
- Penulis
- Penerbit
- Tahun Terbitan
- Bahasa
- Rumusan
- Pengajaran


---

## 使用方法


1. 打开网页


2. 输入书名

例如：

Harry Potter and the Philosopher's Stone


3. 点击 Search


4. 复制资料


5. 粘贴到官方 NILAM / AINS 系统
